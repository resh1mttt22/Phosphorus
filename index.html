The Phosphorus Cipher was not really created; it was found. In the 90s and early 2000s system administrators started to see strange patterns, in the crash logs of old operating systems and local server environments. What looked like data or broken code was actually a secret language.

This language was made by computer experts to get around network monitors. They wanted to send messages that could not be detected. So they hid them as broken text. This way the messages could be sent without being noticed.

The Phosphorus Cipher was almost lost when software got better.. Now it has been carefully rebuilt. This translator helps to bring those old terminal messages. It lets us see the hidden communications again. The Phosphorus Cipher is a way to decode these messages.
